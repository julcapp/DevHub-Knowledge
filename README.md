# DevHub Knowledge

**DevHub Knowledge** — официальный репозиторий инженерной памяти проекта DevHub.

Здесь фиксируются продуктовые решения, архитектурные принципы, стандарты разработки, требования, ADR, инженерные поставки и знания, необходимые для долгосрочного развития DevHub.

## Назначение

Репозиторий является единым источником истины для:

- продуктовой стратегии DevHub;
- архитектурных решений;
- инженерных стандартов;
- требований;
- терминологии;
- реестра решений;
- будущей базы знаний AI Hub.

## Основной принцип

Любая утверждённая договорённость по проекту DevHub должна быть зафиксирована в GitHub.

Чат используется для обсуждения. GitHub используется для официальной фиксации.
