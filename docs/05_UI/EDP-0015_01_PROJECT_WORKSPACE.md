# EDP-0015.01 — Project Workspace

**Код:** UI-0003  
**Версия:** 0.1.0-alpha  
**Статус:** Approved Concept

## Назначение

Project Workspace — персональная рабочая область пользователя после входа в DevHub.

Цель — заменить набор отдельных экранов единым рабочим центром.

---

# Главный экран

```
Здравствуйте, Александр

Рабочая область

--------------------------------
Мои проекты

[ DevHub ]
[ Lk_uni ]
[ Atom QMS ]
--------------------------------

Последние действия

• Создан документ
• Изменена роль
• Новый участник

--------------------------------

Мои задачи

3 активные

--------------------------------

AI помощник

[Задать вопрос]
```

---

# Разделы Workspace

## 1. Мои проекты

Показывает:

- проекты пользователя;
- роль в проекте;
- статус;
- последние изменения.

## 2. Документы

Показывает:

- последние документы;
- ожидающие проверки;
- созданные пользователем материалы.

## 3. Задачи

Показывает:

- назначенные задачи;
- сроки;
- приоритеты.

## 4. Активность

История:

- входы;
- изменения;
- комментарии;
- релизы.

## 5. AI Hub

Персональная точка взаимодействия с ИИ:

- вопросы по проектам;
- поиск знаний;
- подготовка документов;
- анализ изменений.

---

# Безопасность

В Workspace доступны:

- текущая сессия;
- устройства;
- повторная авторизация;
- настройки безопасности.
