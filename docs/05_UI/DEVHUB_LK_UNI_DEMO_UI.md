# DevHub + Lk_uni Demo UI Concept

**Код:** UI-0001  
**Версия:** 1.0.0-alpha  
**Статус:** Approved Concept

## Назначение

Документ описывает первый вариант пользовательского интерфейса связки DevHub и Lk_uni.

Цель — показать пользователю понятный путь от входа до работы в проекте.

---

# Экран 1. Вход

```
+--------------------------------+
|             DevHub             |
|                                |
|  Управляйте проектами          |
|  знаниями и разработкой        |
|                                |
|   [ Войти через Lk_uni ]       |
|                                |
|   [ Создать аккаунт ]          |
+--------------------------------+
```

## Логика

Кнопка входа передает пользователя в Lk_uni.

DevHub не хранит пароль пользователя.

# Принципы интерфейса

1. Пользователь работает с понятными действиями.
2. Сложность авторизации скрыта внутри Lk_uni.
3. Роли и права понятны человеку.
4. Все важные действия имеют историю.
5. Безопасность не мешает обычной работе.
