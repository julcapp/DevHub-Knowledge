# Role Permission Matrix

**Статус:** Draft for EDP-0013.01

## Назначение

Матрица определяет базовые права пользователей DevHub.

## Роли

| Действие | Администратор | Владелец проекта | Руководитель | Архитектор | Разработчик | Аналитик | Наблюдатель |
|-|-|-|-|-|-|-|-|
| Просмотр проектов | Да | Да | Да | Да | Да | Да | Да |
| Создание проекта | Да | Да | Нет | Нет | Нет | Нет | Нет |
| Удаление проекта | Да | Да | Нет | Нет | Нет | Нет | Нет |
| Управление участниками | Да | Да | Да | Нет | Нет | Нет | Нет |
| Изменение ролей | Да | Да | Ограниченно | Нет | Нет | Нет | Нет |
| Просмотр кода | Да | Да | Да | Да | Да | Да | Да |
| Изменение кода | Да | Да | Да | Да | Да | Нет | Нет |
| Архитектурные решения | Да | Да | Да | Да | Предложение | Нет | Нет |
| Управление AI настройками | Да | Да | Ограниченно | Да | Нет | Нет | Нет |
| Просмотр журнала | Да | Да | Да | Да | Ограниченно | Ограниченно | Нет |

## Принцип

Права назначаются не пользователю напрямую, а через роль проекта.

```
User
 ↓
ProjectMember
 ↓
Role
 ↓
Permission
```
