## Назначение

Глоссарий фиксирует единую терминологию DevHub.

Пользовательский интерфейс должен использовать понятные русские термины.

## Термины

| Технический термин | Пользовательский термин | Описание |
|---|---|---|
| Repository | Проект | Совокупность файлов, кода, документов и истории разработки |
| Workspace | Рабочая область | Каталог или среда, где находятся проекты |
| Dashboard | Рабочий центр | Экран, который помогает выполнить действия |
| Fork | Создать свою копию | Создать независимую копию проекта для дальнейшей работы |
| Clone | Загрузить проект | Скопировать проект на компьютер |
| Pull | Получить изменения | Загрузить последние изменения из удалённого источника |
| Push | Отправить изменения | Передать локальные изменения в удалённый источник |
| Commit | Зафиксировать изменения | Сохранить изменения в истории проекта |
| Branch | Ветка разработки | Отдельная линия разработки |
| Merge | Объединить изменения | Объединить изменения из разных веток |
| Release | Выпуск версии | Подготовленная версия продукта |
| Issue | Задача | Работа, ошибка или запрос на изменение |
| Pull Request | Запрос на включение изменений | Предложение включить изменения в основную версию |
| AI Provider | Поставщик ИИ | Сервис или модель, через которую DevHub получает ответы ИИ |
| AI Hub | Центр ИИ | Подсистема подключения ИИ и агентов |
| Project Passport | Паспорт проекта | Карточка проекта с ключевой информацией и действиями |
| Action Center | Центр действий | Область интерфейса, где пользователь выполняет действия над проектом |
| Event Log | Журнал событий | Запись действий пользователя, системы и ИИ |
| Engineering Memory | Инженерная память | Совокупность решений, документов, событий и знаний проекта |
| ADR | Архитектурное решение | Документ, фиксирующий важное архитектурное решение |
| EDP | Инженерная поставка | Завершённый пакет изменений кода, документов и тестов |
| FP | Foundation Pack | Поставка базовой документации и стандартов |

## Правило перевода

Если термин можно заменить понятным русским словом без потери смысла, пользовательский интерфейс использует русский термин.

Англоязычный термин может отображаться в скобках только в профессиональном режиме.
